# RETNI – Controle Financeiro

Sistema web desenvolvido para auxiliar no controle financeiro pessoal, permitindo o registro e acompanhamento de transações de forma simples e organizada.

## 📌 Sobre o projeto

O RETNI foi criado com o objetivo de facilitar a visualização de entradas e saídas financeiras em um único lugar, oferecendo uma interface limpa e funcional para o dia a dia.

## 🚀 Tecnologias utilizadas

* Next.js
* React
* TypeScript
* Tailwind CSS

## ⚙️ Funcionalidades

* Cadastro de transações financeiras
* Listagem de movimentações
* Interface organizada e responsiva
* Visualização clara de entradas e saídas

## ▶️ Como executar o projeto

Clone o repositório:

```bash

```

Acesse a pasta do projeto:

```bash
cd retni
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Abra no navegador:

```
http://localhost:3000
```

## 🌐 Deploy

O projeto está disponível online através da plataforma Vercel.
https://controle-financeiro-retni.vercel.app/

## 📁 Estrutura do projeto

* `/app` → páginas da aplicação
* `/components` → componentes reutilizáveis
* `/hooks` → funções customizadas
* `/lib` → utilidades e configurações
